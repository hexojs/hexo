使用

安裝

執行以下指令。 

npm install <plugin-name>

啟用

在_config.yml的plugins欄位中加入外掛名稱。 

plugins:
- plugin-one
- plugin-name

停用

從_config.yml的plugins欄位中移除外掛名稱。 

plugins:
- plugin-one

更新

執行以下指令。 

npm update

移除

執行以下指令，在移除前別忘了先停用外掛。 

npm uninstall <plugin-name>

Generator

Generator 用於生成靜態檔案。

  • hexo-generator-feed - RSS Feed
  • hexo-generator-sitemap - Sitemap

Renderer

Renderer 用於處理特定類型的檔案。

  • hexo-renderer-coffeescript - CoffeeScript
  • hexo-renderer-haml - Haml
  • hexo-renderer-jade - Jade
  • hexo-renderer-less - Less

Helper

Helper 為模版檔案中使用的函數。

Deployer

Deployer 用於佈署檔案。

Processor

Processor 用於處理原始檔案。

Tag

Console

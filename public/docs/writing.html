Writing

Post

Create

Execute the following command. 

hexo new_post <title>

New post will saved at source/_posts/title.md in the root directory with the following content. 

---
layout: post
title: title
date: YYYY-MM-DD HH:mm:ss
comments: true
tags:
---

Configure

文章最前面有一段用---包裹的區塊，稱為 YAML Front Matter。你可使用 YAML 格式設定文章。

以下是預設的設定內容，你可隨自己喜好增減內容。

  • layout - 文章布局
  • title - 文章標題
  • date - 文章的發佈日期（格式為 YYYY-MM-DD HH:mm:ss）
  • comments - 開啟此文章的留言功能
  • tags - 文章標籤

Category

分類為文章與 source/_posts 的相對位址，具有階層性。

For example:

  • source/_posts/title.md - 無分類
  • source/_posts/Fruits/title.md - Fruit
  • source/_posts/Fruits/Apple/title.md - Fruit, Apple

Tag

# Single Tag
tags: Apple

# Multiple Tags
tags: [Apple, Banana]

tags:
- Apple
- Banana

Page

Create

Execute the following command. 

hexo new_page <title>

New page will saved at source/title/index.md in the root directory with the following content. 

---
layout: page
title: title
date: YYYY-MM-DD HH:mm:ss
comments: true
---

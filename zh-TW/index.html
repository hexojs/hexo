快速、簡單且功能強大的 Node.js 網誌框架。
Tweet

功能

  • 不可思議的快速 ─ 只要一眨眼靜態檔案即生成完成
  • 支援 Markdown
  • 僅需一道指令即可佈署到 GitHub Pages 和 Heroku
  • 已移植 Octopress 外掛
  • 高擴展性、自訂性
  • 相容於 Windows, Mac & Linux

安裝

npm install -g hexo

更新

npm update -g

開始

建立專案： 

hexo init project && cd project

建立新文章： 

hexo new_post title

生成靜態檔案： 

hexo generate

啟動伺服器： 

hexo server

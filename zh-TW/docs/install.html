安裝

在安裝 Hexo 之前，必須先安裝 Git 和 Node.js。

目錄

  • Windows
  • Mac
  • Linux

Windows

Git

下載 msysgit 並執行即可完成安裝。

Node.js

在 Windows 環境下安裝 Node.js 非常簡單，僅須 下載 安裝檔案並執行即可完成安裝。

Hexo

利用 npm 即可安裝。 

npm install -g hexo

Mac

Git

Mac 已內建 Git，雖然版本較舊，不過還是能用，如果你想要使用較新版的 Git 的話，可參考以下方式安裝。

  1. 使用 Homebrew。

    brew install git

  2. 使用 MacPorts。

    sudo port install git-core

  3. 使用 安裝程式。

如果出現錯誤情況，可能是尚未安裝編譯工具，請至 App Store 下載 Xcode。

Node.js

Mac 環境下有很多安裝方式可供選擇，以下使用 nvm 來安裝。

執行以下指令。 

git clone git://github.com/creationix/nvm.git ~/nvm

安裝完畢後，在~/.bash_profile或~/.bashrc加入以下內容並重開一個新的終端機視窗。 

. ~/nvm/nvm.sh

之後即可使用 nvm 來安裝 Node.js 了，編譯可能需要花些時間，請耐心等候。 

nvm install v0.8.14
nvm use v0.8.14
nvm alias default v0.8.14

Hexo

利用 npm 即可安裝。 

npm install -g hexo

Linux

Git

Ubuntu, Debian: 

sudo apt-get install git-core

Fedora, Red Hat, CentOS: 

sudo yum install git-core

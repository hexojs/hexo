伺服器

內建伺服器

Hexo 使用 Connect 作為靜態檔案的伺服器。

編輯_config.yml中的port欄位調整伺服器的連接埠。 

port: 4000

執行以下命令啟動伺服器，按下Ctrl+C關閉伺服器。 

hexo server

記錄器

編輯_config.yml中的logger欄位啟動記錄器。編輯logger_format欄位可調整記錄的顯示內容，參考 Connect 以獲得更多資訊。 

logger: true
logger_format:

Pow

Pow 是由 Node.js 所建立的 Mac 環境專用的零配置 Rack 伺服器，不過它也能用於處理一般的靜態檔案。

安裝

執行以下命令即可完成安裝。 

curl get.pow.cx | sh

使用

在~/.pow建立連結即可使用。 

cd ~/.pow
ln -s /path/to/myapp

完成後，網站即會出現在http://myapp.dev，網址根據連結名稱而有所不同。

參考 Pow 以獲得更多資訊。

遷移

目錄

  • RSS
  • Jekyll / Octopress

RSS

首先，執行以下指令安裝 RSS Migrator 外掛，並在_config.yml中的plugins欄位加入hexo-migrator-rss。 

npm install hexo-migrator-rss

安裝完畢後，執行以下指令即可，source參數可為檔案路徑或網址。 

hexo migrate rss

Jekyll / Octopress

將source/_posts內的檔案拷貝至 Hexo 的source/_posts內，並將文章內的categories屬性修改為tags即可。

WordPress

首先，執行以下指令安裝 WordPress Migrator 外掛，並在_config.yml中的plugins欄位加入hexo-migrator-wordpress。 

npm install hexo-migrator-wordpress

安裝完畢後，執行以下指令即可，source參數可為檔案路徑或網址。 

hexo migrate wordpress
